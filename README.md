# 🛰️ Olist E-Commerce Intelligence: Master Command Center

A high-fidelity, executive Business Intelligence suite built for the Brazilian Olist ecosystem. This project transforms fragmented e-commerce data into a "Tactical Command Center" powered by **AWS RDS**, **dbt**, and **Machine Learning**.

![Tech Stack](https://img.shields.io/badge/Stack-AWS_|_dbt_|_Streamlit_|_ML-blue?style=for-the-badge)

## 💎 Project Highlights
- **Single-Page Tactical HUD**: A high-density "Master Command" grid built with custom CSS for a premium, tactical aesthetic.
- **Deep Analytical Marts**: Powered by a robust **dbt (Data Build Tool)** transformation layer hosted on **AWS RDS**.
- **Real-Time AI Strategy**: Integrated **Random Forest** churn prediction engine with a live-responsive "Matrix Scanner" for risk simulation.
- **The "Big Six" Intelligence Modules**:
    1. **Revenue Composition**: Spline-area analysis of category-level GMV trends.
    2. **Logistics Friction**: Identifying supply chain bottlenecks via high-latency cluster mapping.
    3. **Customer Universe**: RFM-based sunburst segmentation of LTV and churn risk.
    4. **Geospatial Power**: Ranking geographic revenue engines across Brazil.
    5. **Seller Contribution**: Analyzing concentration risk across tiered seller cohorts.
    6. **Strategy Forecaster**: A live AI simulation engine for tactical retention planning.

## 🛠️ Technical Architecture
- **Database**: PostgreSQL on **Amazon Web Services (AWS) RDS**.
- **Transformation**: **dbt** models for modular, scalable analytics.
- **Frontend**: **Streamlit** with bespoke CSS injection for "Command Center" depth.
- **Visuals**: **Plotly Express** & **Graph Objects** for interactive, high-fidelity charts.
- **Modeling**: **Scikit-learn** Random Forest Classifier for predictive churn scoring.

## 🚀 Getting Started
1. **Clone the Repo**: `git clone <your-repo-link>`
2. **Environment**: Setup `.env` with AWS RDS credentials.
3. **Launch**:
   ```bash
   source venv/bin/activate
   streamlit run app.py
   ```
